## Visão Geral

### O desafio

Os usuários devem ser capazes de:

- Ver o layout otimizado do componente dependendo do tamanho da tela do dispositivo
- Ver os links de compartilhamento de mídia social ao clicar no ícone de compartilhamento
- Experiência responsiva que se adapta entre dispositivos móveis (375px) e desktop (768px+)
